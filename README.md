AccuKnox AI/ML Trainee Assignment

This project demonstrates a complete data pipeline and backend system using Python.

Features:
- Flask REST API for student dataset
- Fetching data from API using requests
- Data processing using pandas
- Visualization using matplotlib
- CSV data handling
- SQLite database integration

Project Structure:
- student_api.py → Flask API server
- fetch_scores_from_api.py → API client, data processing and visualization
- api_to_sqlite_books.py → Fetch books from API and store in SQLite
- csv_to_sqlite.py → Import CSV data into SQLite
- student_scores.csv → Dataset
- names_emails.csv → Sample user dataset
